Por qué funciona la data propia
0P
Base CRM Propia
Emails y datos de clientes reales de L'Oréal. Usuarios que ya compraron, se registraron o interactuaron con la marca. Mayor intención de compra.
1P
Visitantes del Sitio
Usuarios que visitaron páginas de producto, añadieron al carrito o vieron contenido. Señales de comportamiento real. Mid/lower funnel.
FR
Fishing Rules
Cruces entre marcas con afinidad comprobada. Ej: visitantes de Kiehl's hacia SkinCeuticals. Amplía alcance con calidad.
2P
Audiencias Externas
Segmentos por intereses de terceros. Útiles para escalar y nutrir la base 1P. Awareness, no conversión.

Best Practices · CPD
Awareness como Objetivo Principal
  • Priorizar 2P para maximizar alcance y cobertura en lanzamientos
  • CPM como KPI principal — no CTR ni conversión directa
  • Cada impresión es una oportunidad de construir marca
Audiencias 1P para Amplificar
  • Activar 1P progresivamente en lower funnel para eficiencia
  • Los usuarios que enganchan en awareness pasan a ser data 1P
  • AON activo para sostener volumen de datos propios
Modelo de Alimentación
  • 2P captura → 1P activa → 0P convierte
  • CPD alimenta la base 1P de todo el ecosistema L'Oréal
  • Awareness hoy = intención mañana
